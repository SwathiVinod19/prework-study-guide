HTML

  • HTML elements build structure of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS determines the style of a webpage and makes it visually appealing.
  • CSS also makes the website simpler, more organized.
  • There are 3 ways to style a webpage using CSS:Inline, Internal and External CSS.
  • When a developer wants to style an element directly in the HTML file, they use Inline CSS. They're useful for quick and permanent changes but less flexible.
  • Example of inline CSS:using the style attribute to change the colour of a single heading.
  • Internal CSS contains CSS rules for the webpage in the head section of the HTML file.
  • Most common way to style a webpage is using External CSS.
  • Using external CSS we linked the file using a link elemet placed in the head of the HTML file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an elemet.
